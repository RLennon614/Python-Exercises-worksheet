Day 1: Digit Extraction & Manipulation

The core challenge was to manipulate the digits of a number without converting the number to a string. This demonstrates understanding of Python's fundamental operators:

Problem: Find the sum of the digits of a three-digit number.

Concepts: Using the modulus operator (%) to isolate the last digit, and floor division (//) to drop the last digit.

Result: A program that can accurately calculate complex numerical properties using pure arithmetic logic.

Day 2: Conditional Logic & Flow Control

This section verifies the ability to build programs that make decisions based on different inputs:

Problem: Create a program that determines the user's age category (Child, Teenager, Adult) based on input.

Concepts: Nested if/elif/else statements to define clear execution paths.

Result: Robust programs that handle multiple scenarios and ensure only one outcome is achieved per input.
